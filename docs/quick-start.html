Anonymiseur Action Telecom

Démarrage rapide — v1.0.0

À quoi ça sert ?

Avant de coller un extrait de log, ticket, email ou capture dans un assistant IA (ChatGPT, Claude…), passe-le dans cet outil. Il remplace les données sensibles (emails, IP, noms, IBAN, CB, numéros de sécu…) par des marqueurs. Aucune donnée ne sort de ton poste.

Le mapping JSON contient les vraies valeurs : ne le partage pas, ne le mets pas dans un mail. Garde-le sur ton poste.

Lancement

  1. Menu Démarrer → Action Telecom → Anonymiseur.
  2. Le navigateur s'ouvre tout seul sur http://127.0.0.1:7777.
  3. La fenêtre console noire qui s'affiche : ne pas la fermer, c'est le moteur. Fermer = arrête l'outil.

Workflow standard

1. Anonymiser

  1. Glisse un fichier (.txt, .log, .pdf, image…).
  2. Clique Anonymiser.
  3. Copie le texte anonymisé → colle-le dans l'IA.
  4. Télécharge le mapping.json (à garder si tu veux restaurer plus tard).

2. Dé-anonymiser

  1. Récupère la réponse de l'IA.
  2. Onglet Dé-anonymiser.
  3. Colle la réponse + fournis le mapping.json.
  4. Récupère le texte avec vraies valeurs.

Problèmes fréquents

  • Le navigateur ne s'ouvre pas : ouvre-le manuellement sur http://127.0.0.1:7777. Si le port est occupé, regarde la console (le port effectif y est affiché).
  • "Tesseract non détecté" : réinstalle l'Anonymiseur. Tesseract est embarqué dans l'installeur.
  • PDF refusé "trop volumineux" : limite 50 pages pour l'OCR. Pour PDF natif (avec texte sélectionnable), pas de limite.
  • L'outil n'a pas remplacé un nom : la détection NER n'est pas parfaite, surtout sur prénoms courts ou organisations rares. Relis le texte anonymisé avant envoi.

Données et confidentialité

  • 100 % local. Aucune connexion sortante. Aucune télémétrie.
  • Uploads en mémoire uniquement. Aucun fichier client n'est écrit sur disque.
  • Logs techniques (sans contenu) : %LOCALAPPDATA%\ActionTelecom\Anonymiseur\logs\
