⚙ Configuration Tesseract OCR

Tesseract est requis pour analyser les images et les PDF scannés. Il s'installe séparément (gratuit, open-source).

  1. Téléchargez l'installeur Windows :
    https://github.com/UB-Mannheim/tesseract/wiki
  2. Lancez l'installeur — cochez "French" dans les langues.
  3. Notez le dossier d'installation (ex : C:\Program Files\Tesseract-OCR).
  4. Collez le chemin complet vers tesseract.exe ci-dessous.
